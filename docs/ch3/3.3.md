## 3.3 참조 타입의 특성
[예제3-10][3-10]

### 3.3.1 객체 비교
동등 연산자 `==` 를 사용하여 두 객체를 비교하면 프로퍼티의 값이 아닌 참조값을 비교한다.

[예제3-11][3-11]

### 3.3.2 참조에 의한 함수 호출 방식
- 기본 타입의 경우 **값에 의한 호출<sup>Call By Value</sup>** 복사된 값이 전달
- 참조 타입의 경우 **참조에 의한 호출<sup>Call By Reference</sup>** 객체의 참조값이 전달
