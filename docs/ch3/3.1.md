- 데이터 타입
  - 기본 타입
    - 숫자(Number)
    - 문자열(String)
    - 불린값(Boolean)
    - undefined
    - null
  - 참조 타입
    - 객체(Object)

## 3.1 ㅈㅏㅂㅏ스크립트 기본 타입
기본 타입은 **숫자**, **문자열**, **불린값**, **null**, **undefined** 타입이 있다.
